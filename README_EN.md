## Installation

```bash
pip install --upgrade pip
pip install dbgone
```

## Features

### dbgone.config

  Configuration management functionality

#### dbgone.optuna

  Optuna management functionality

### dbgone.torch

  PyTorch model management functionality

> Note: Requires extra installation:
```bash
pip install dbgone[torch]
```
